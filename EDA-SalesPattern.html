Descobrindo Padres em Vendas

Objetivo

Analisar um dataset de vendas reais para desvendar os fatores que impulsionam o desempenho ao longo do tempo, combinando anlise temporal avanada, segmentao detalhada e automao, com o intuito de extrair insights acionveis e preparar uma base robusta para modelos preditivos de Machine Learning.

Razo

Escolhi este projeto como o pice da minha jornada em EDA porque vendas so o corao de muitos negcios, e entender seus padres um desafio real e universal. Quis ir alm do bsico, explorando como o tempo, os clientes e as condies afetam os resultados, e transformar dados brutos em uma histria que no s informe, mas tambm inspire solues prticas e tecnolgicas.

Temporal

In[48]:
# Function to update df_outliers and df_typical based on df
def update_subsets(df):
    Q1 = df['Sales'].quantile(0.25)
    Q3 = df['Sales'].quantile(0.75)
    IQR = Q3 - Q1
    upper_bound = Q3 + 1.5 * IQR
    df_outliers = df[df['Sales'] > upper_bound].reset_index(drop=True)  # Outliers subset
    df_typical = df[df['Sales'] <= upper_bound].reset_index(drop=True)  # Typical subset
    return df_outliers, df_typical
